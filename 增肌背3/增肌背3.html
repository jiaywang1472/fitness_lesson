理论课


背部肌群的训练需要从三个大方向着手，分别是垂直拉、水平拉和单关节练习。

  • 垂直拉（坐姿下拉、引体向上等练习）主要目标肌群是背阔肌，这类练习可增加背宽度，打造腰背比，塑造躯干两侧线条

  • 水平拉（坐姿划船、单臂划船等划船类练习）主要目标是菱形肌，这类练习可增加背的立体感、使背部线条凸显出来

  • 单关节练习的主要目的是“修补”，针对性的去孤立刺激较弱的背部肌群，使背部肌群更协调



热身部分

01

泡沫轴放松背阔肌

训练要求：

  • 腋窝靠后侧位置与泡沫轴相接触，小幅度上下移动（幅度约5cm），若肌群紧张会有明显痛点

  • 30s/侧/组，完成1组


02

泡沫轴放松上背部（选做）

训练要求：

  • 选择材质较软的泡沫轴（可在泡沫轴表面裹毛巾/毯子，增加柔软度），避免硌伤胸椎

  • 在胸椎段小幅度上下移动（泡沫轴最外沿不可移动到颈椎下方），避免触碰颈椎

  • 30s/组，完成1组



激活部分


提高控制肩胛骨的能力，更好的服务于背部训练

（肩胛骨控制能力较强可不做）

01

肩胛骨激活练习1

训练要求：

  • 准备姿势：保持双臂前平举、肘关节微屈（全程不要改变肘关节的弯曲角度）

  • 向心收缩：肩胛骨主动后缩夹背，带动手臂往后侧移动

  • 离心收缩：肩胛骨往两侧展开，带动手臂往前伸直

  • 10-15次/组，完成1组


02

肩胛骨激活练习2

训练要求：

  • 准备姿势：保持双臂侧平举、肘关节微屈（全程不要改变肘关节的弯曲角度）

  • 向心收缩：肩胛骨主动后缩夹背，带动手臂往中间小幅度移动

  • 离心收缩：肩胛骨往两侧展开，带动手臂小幅度往两侧移动

  • 注意：侧面观，尽量避免手臂前后摇晃（20cm以内）

  • 10-15次/组，完成1组


03

肩胛骨激活练习3

训练要求：

  • 准备姿势：双臂举过头顶，肘关节微屈（全程不要改变肘关节的弯曲角度）

  • 离心收缩：肩胛骨主动抬高，带动手臂往上小幅度移动

  • 向心收缩：肩胛骨主动压低，带动手臂往下小幅度移动

  • 10-15次/组，完成1组



正式训练


  1. 在力量允许的前提下逐步增加负重，无法增加负重时，应选择合适的重量，尽量完成多次数

  2. 当发力感不好时，优先降低负重，放缓动作节奏，集中注意力在目标肌群上，寻找发力感

  3. 拍摄视频以规范动作

  4. 以下所有练习组间休息30-90秒

01

器械坐姿下拉

可改为备选练习：器械引体向上

点击跳转视频讲解（A视频库）


任何下拉类固定器械皆可，以下提供其他器械示范 ⬇️

目标肌群：

  • 背阔肌（主）

  • 肱二头肌、肩背部其他肌群（辅）    

动作步骤：

  • 准备姿势：双手握住手柄，坐在训练凳上，膝盖抵住上方的定位垫，稳定住身体

  • 向心收缩：背部两侧的背阔肌发力协同手臂肌群，将器械下拉，直至腕关节和肩关节在水平面几乎齐高

  • 顶峰收缩：拉到最低点时停顿片刻，感受背部肌群的收缩发力

  • 离心收缩：背部肌群发力，控制着缓慢匀速复位

  • 呼吸：向心吐气，离心吸气

易错点/注意事项：

  • 避免过度反弓、下背部借力，轻微反弓是允许的

  • 因目标肌群主要是背阔肌而非菱形肌，所以下拉时无需过于用力夹紧肩胛骨（不似划船类练习），只需轻微发力配合肩胛骨下旋

完成方式：

第一组 20RM

第二组 15RM

第三组 12RM

第四组 10RM（选做）

第五组 10RM（选做）



02

俯身杠铃划船

点击跳转视频讲解（B视频库）

目标肌群：

  • 菱形肌和斜方肌中下部（主）

  • 其他背部肌群、肱二头肌以及肩部肌群（辅）    

动作步骤：

  • 准备姿势：俯身屈膝屈髋，站距与髋同宽；小腿垂直于地面，躯干和地面夹角约45度；双手持杠铃自然下垂，肩胛骨放松展开

  • 向心收缩：背部发力收紧肩胛骨，带动肘部往身体后侧划动，小臂应保持垂直于地面或略微向后侧倾斜

  • 顶峰收缩：当杠铃杠触碰到下腹部时，停顿片刻

  • 离心收缩：缓慢匀速下放，如此反复

  • 向心吐气，离心吸气

易错点/注意事项：

  • 初学者负重不宜过重容易借力，注意顶峰收缩的停顿，动作节奏不宜过快

  • 避免耸肩和摇晃躯干借力

  • 下背部可轻微反弓，⚠️ 不能龟背

  • 顶峰收缩时充分的夹紧肩胛骨，主动发力肌群是背部而不是手臂，避免手臂发力主动提拉

  • 背部夹紧即可，肘关节不要提拉过高，容易增加肩关节压力

完成方式：

  • 10-20RM/组，完成3-4组



03

V柄坐姿下拉

点击跳转视频讲解（B视频库）

目标肌群：

  • 背阔肌、大圆肌（主）；

  • 斜方肌、肱二头肌（辅）

动作步骤：

  • 准备姿势：该练习可在固定器械、龙门夹绳索上完成；双手握住V柄，若无V柄可使用两个普通手柄组成V柄

  • 向心收缩：背部发力将手柄匀速下拉，用时肩胛骨略微发力下沉、后缩；

    动作最低点时，腕关节和肩关节在水平面平行

  • 离心收缩：复位时，避免肘关节完全伸直锁死，使背部肌群和肩胛骨充分伸展开，肩关节不要完全松懈

  • 呼吸：向心吐气，离心吸气 

易错点/注意事项：

  • 下拉时肩胛骨要主动配合发力，积极参与，否则手臂肌群和肩部肌群发力比例会增加

  • 避免运动幅度过大   

  • 下背部可略微反弓，但需避免过度反弓

完成方式：

第一组 20RM

第二组 15RM

第三组 12RM

第四组 12RM（选做）

第五组 10RM（选做）



04

单臂哑铃划船

点击跳转视频讲解（A视频库）

🗣 教练点评：

1. 动作顶峰时，双肩连线尽量不要大幅度的旋转啊，会更接近平行地面。

2. 我们只要肩胛骨主动后收夹背就行，胸椎可以不用大幅度的旋转。

3. 平息往下降的时候，双肩连线可以去倾斜，让背部肌群更充分的拉伸开。

4. 支撑手的肘关节，注意保持微屈，不要完全伸直锁死。

目标肌群：

  • 背阔肌和大圆肌（主）

  • 肱二头肌、其他肩背肌群（辅）

动作步骤：

  • 准备姿势：假设左侧完成训练，则将你的右侧膝盖和手支撑在平板凳上；支撑侧肘关节和支撑侧的膝关节保持微屈，避免伸直锁死；使脊柱处于中立位，收腹提臀

  • 向心收缩：背部发力带动肘关节往后滑动（需主动后缩肩胛骨），发力时不要想着用手臂发力把重物提起，而是把注意力集中在背部

  • 离心收缩：背部发力，控制着匀速缓慢复位，使背部肌群充分被拉伸、打开肩胛骨

  • 呼吸：向心吐气，离心吸气

易错点/注意事项：

  • 摆正骨盆，避免侧倾侧旋

  • 向心收缩时，手肘避免外展，应靠近躯干上下运动，但无需刻意贴紧

  • 动作顶峰，使大臂平行往上移动挤压背部，而非挤压肩部，所以动作最高点时肘关节不要高于肩关节太多

  • 运动过程中，下背可轻微反弓，避免龟背

  • 保持小臂时刻垂直于地面

完成方式：

第一组 15RM

第二组 12RM

第三组 12RM

第四组 12RM（选做）

第五组 10RM（选做）



05

仰卧直臂哑铃上拉

目标肌群：

  • 背阔肌（主）

  • 胸部肌群、大圆肌、肱三头肌（辅）

动作步骤：

  • 准备姿势：背对躺椅，使双肩和躺椅相互接触，起到支撑所用；臀部保持悬空，小腿接近垂直地面，双脚站距与肩同宽；

    可手持哑铃完成准备姿势，若负重较重可寻求伙伴帮助，或将负重物放置在身侧的躺椅上，方便拿取；

    双手交叠持哑铃，虎口抵住哑铃手柄，手指扣住哑铃一端，确保动作过程中的安全稳定；手臂伸直，使哑铃置于胸口正上发，肘关节保持微微弯曲

  • 离心收缩：背部发力做退让性工作，使哑铃缓慢往脑后方向移动，同时臀部缓慢下沉，辅助背阔肌拉长，动作全程感受背阔肌的拉长拉伸感，动作最低点稍顿片刻，缓慢复位

  • 向心收缩：复位速度可比离心收缩过程略快，感受背部肌群缩短、发力

  • 呼吸：离心吸气，向心吐气

    动作节奏：离心慢，向心略快

易错点/注意事项：

  • 动作全程固定住肘关节，保持微微弯曲，避免伸直锁死或过度弯曲

  • 离心收缩速度要慢，并且匀速进行，向心收缩避免使用爆发力

  • 臀部无需下移过多

完成方式：

第一组：20RM

第二组：15RM

第三组：12RM

第四组：12RM（选做）

第五组：10RM（选做）



备选动作


  1. 正式训练中，目标肌群相同的练习可用备选代替

  2. 正式训练中所有练习做完，认为训练量偏小，可追加备选动作（不推荐）

01

器械引体向上

🗣 教练“说人话”点评：

1. 器械“嫌你矮”： 正常拉到最高点时，肩膀和手腕应该差不多高。但现在你手腕比肩膀高太多，说明这个器械的设计更适合高个子男生，对你来说“太长了”。

2. 两个自救方案：

• 方案 A（加垫子）： 找个厚一点的拳击垫垫在膝盖下，把身体撑高一点。这样你往上拉的空间就变大了，动作才能做完整。
• 方案 B（直接换动作）： 如果健身房没垫子，千万别死磕这个器械。直接去改做“坐姿下拉”，效果是一样的，没必要在不合适的机器上浪费力气。

3. 核心逻辑： 不要强行适应器械。既然你正忙着完善自己的课程系统，记录下这些器械适配的小知识，能让你在不同的健身房都能找到最高效的训练方式。

目标肌群：

  • （主）背阔肌；大圆肌；

  • （辅）三角肌后束；肱二头肌

动作步骤：

  • 准备姿势：双腿跪在支撑垫上，双手握住器械，以动作顶峰小臂垂直于地面为基准，选择握距；

    负重片数值越大，训练时难度越低（初学者可选择接近略小于自身体重的重量）

  • 向心收缩：背部肌群发力带动手臂将躯干往上拉起；

    过程中躯干略微后仰，肩胛骨可轻微后缩夹背

  • 顶峰收缩：拉至最高点时，肩关节在水平面与腕关节齐高/略低于5cm（这个幅度区间内都是可以的）；

    如果因身高原因，无法拉到标准幅度，可在膝盖下垫厚度足够的物体（比如瑜伽砖、拳击靶等）

  • 离心收缩：到达动作最高点后稍顿片刻，缓慢匀速下降

  • 呼吸：向心吐气，离心吸气

易错点/注意事项：

  • 避免圆肩驼背（肩往前扣）借力，应略微挺胸，把肩打开；当然也要避免过度后仰借力

  • 需主动调动背部肌群发力，避免手臂肌群参与度过高

  • 离心收缩需积极建立与背部肌群的练习，有控制的缓慢下放，并感受背部肌群的拉伸感

  • 温馨提示：离开器械时一定要小心，利用自身体重，控制支撑垫回到最高点后再缓慢离开垫子；

    否则，你会听到“嘭！”的一声巨响

完成方式：

第一组 15RM

第二组 15RM

第三组 12RM

第四组 10RM（选做）

第五组 10RM（选做）



拉伸部分


能（一定程度）消除疲劳、加速血液循环

帮助目标肌群恢复、防止目标肌群粘连和痉挛

01

拉伸背阔肌

训练要求：

  • 单臂支撑在高度合适的支撑物/墙面上

  • 充分屈肩屈髋，感受背阔肌的拉伸感

  • 身体小幅度侧倾，可增强拉伸感

  • 拉伸时肩关节不适，可使肩部肌群略微发力，避免肩关节过度放松，同时把注意集中在背阔肌上

  • 30s/侧/组，完成1组


02

拉伸肱二头肌

训练要求：

  • 手臂伸直，如图二手腕外旋，使手掌虎口朝下，支撑在墙面

  • 整个躯干扭转向另一侧，略微沉肩，感受肱二头肌肌纤维被拉长

  • 30s/侧/组，完成1组
